# Pixel Ruins

A pixel-art 2D platform adventure built with plain JavaScript and Canvas.

Includes:
- Custom pixel assets (player, enemies, coins, portal, tiles)
- Chiptune-style background music
- Sound effects (jump, coin, hit, portal, win)
- Three handcrafted levels with progression

## Run

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Controls

- Move: `A` / `D` or `Arrow Left` / `Arrow Right`
- Jump: `W`, `Arrow Up`, or `Space`

Click **Enable Audio** to start music and effects.

## Test

```bash
npm test
```
